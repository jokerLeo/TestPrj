# ray分布式敏捷开发

#### 介绍
使用springboot+mybatisplus+dubbo的分布式敏捷开发项目，持续更新中...

#### 软件架构
基于springboot+mybatisplus+dubbo的分布式敏捷开发项目，包含以下功能：
- 1.一键生成entity,dao,service,serviceImpl,controller,同时service满足dubbo配置，可供其他项目调用
- 2.基于docker-compose的mysql主从一键化搭建
